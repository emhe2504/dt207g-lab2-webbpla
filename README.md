Arbetsloggen är en webbplats där användaren kan se tidigare arbetserfarenheter som hämtas från ett API. Dessa presenteras som listor på startsidan. 
Det går även att radera kurser på startsidan och då raderas de även från API:et. Under fliken "Lägga till" går det att lägga till arbetslivserfarenheter som kommer hamna i en lista på startsidan, 
med arbetserfarenheten kommer även läggas till i API:et. Utöver detta finns en Om-sida som presenterar lite tankar och information om uppgiften.
